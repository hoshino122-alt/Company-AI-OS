# Company AI OS Production System
# Production Bible

Version : 0.1

Created : DAY29

---

# Philosophy

Company AI OS Production System は、
Company AI OSを100日間開発する中で生まれた知識・経験・改善を蓄積し、
未来の制作をより良くするためのシステムである。

Production Systemは完成品ではない。

開発と共に成長し、
気付きをルールへ変え、
ルールをテンプレートへ変え、
テンプレートを資産へ変える。

今日より明日、
明日より未来の制作が少しでも良くなることを目的とする。

---

# Purpose

Production Systemは次の5つを目的とする。

・世界観を統一する
・制作品質を維持する
・制作時間を短縮する
・テンプレートを育てる
・100日後も再利用できる資産を作る

---

# Development Policy

・制作しながらProduction Systemを育てる

・気付いたことはProduction Bibleへ追加する

・テンプレート化できるものはテンプレート化する

・Company AI OSの世界観を維持する

・Production System自身もCompany AI OSと共に成長する

---

# Rule000

Production Systemは完成させるものではない。

実際の制作で得られた知見だけを採用する。

より良い方法が見つかった場合は、
既存ルールを更新することを優先する。
