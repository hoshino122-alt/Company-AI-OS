# DAY17 - ToDo Management

## Overview

DAY17 introduces a ToDo management system for the AI Secretary.

After integrating the calendar, the next step is enabling the AI Secretary to organize daily tasks. The ToDo Manager allows users to add, view, complete, and manage tasks through natural conversations.

Together with the calendar, this forms the foundation of a practical AI productivity assistant.

---

## Features

- Design the ToDo Manager
- Add new tasks
- View task list
- Manage completed tasks
- Connect the AI Secretary to the ToDo Manager
- Test the ToDo workflow

## Learning Points

Task management is one of the core responsibilities of a digital assistant.

By introducing a dedicated ToDo Manager, the AI Secretary can organize daily work, keep track of progress, and provide task information through natural conversations.

This complements the calendar system introduced in DAY16.

## Result

The AI Secretary can now:

- Manage daily tasks
- Add new ToDo items

This establishes the foundation for intelligent task management in Company AI OS.
